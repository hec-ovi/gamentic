Gamentic

Docs & Blueprints

A navigable blueprint of Gamentic, a self-hosted AI dungeon RPG. Each interactive atlas is a pannable, clickable map of one slice of the system; each is paired with an agent-ready markdown twin (the same data, written for a coding agent to load on demand). Click a node for what it reads, generates, writes and connects to.

How to read this. The atlases are self-contained HTML (pan with drag, zoom with the wheel, click a node or edge, ?/H/IO for quick help, Fit to frame, Expand to widen the panel). The guide is the same data as plain markdown so a coding agent can load exactly one file and have everything — a hand-written resolver dispatching to fat, self-contained, git-tracked notes, the same idea as gbrain ("agent brain in markdown"), in this repo's house style. When a doc and the code disagree, the code (and the in-repo INDEX.md maps) win.
